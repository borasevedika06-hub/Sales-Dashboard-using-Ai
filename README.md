# Sales-Dashboard-using-Ai
 The Core Problem Bajaj's CEO and CFO have no single place to look at business performance. The raw data sitting in Excel tells them nothing at a glance. 
Key Business Problems Hidden in the Data

1. Furniture is a trap — high revenue but Tables and Bookcases generate negative profit
2. Discount abuse — orders with 40%+ discounts are almost always loss-making
3. Geography blindspot — Central region underperforms vs West significantly
4. Customer illusion — some high-revenue customers are barely profitable
5. No seasonality awareness — Q4 is consistently the peak but not being capitalized on

The Solution Asked For
Build an interactive executive dashboard that:

1. Tells the story in 3 pages: Big Picture → Profit Leaks → Customer/Ops
2. Has 8 live filters so the CEO/CFO can slice any way they want
3. Surfaces smart insight callouts — no hunting, problems flagged automatically
4. Is boardroom-ready — clean, branded (navy + gold), printable as PDF
